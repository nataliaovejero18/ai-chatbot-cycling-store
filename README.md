# AI Chatbot for Cycling Store

## Project Overview
This project focuses on the design and development of an AI chatbot for an online cycling store.

The chatbot improves user experience by providing:
- Bike recommendations
- Answers to common questions (shipping, returns)
- Help with choosing the correct bike size

## Key Features
- Multi-language support (English & Spanish)
- Personalized recommendations based on user input
- FAQ automation
- Size recommendation system

## Conversation Design
The chatbot uses different workflows:
- Bike Recommendation
- FAQ (shipping, returns, warranty)
- Size Help

It routes users depending on their intent to provide accurate responses.

## Tools
- Voiceflow
- Conversational AI design
- Workflow logic

## Example Interactions
- Recommends bikes based on budget and experience
- Answers shipping questions (2–4 days delivery)
- Suggests bike size based on height

## What I learned
This project helped me understand how AI can be applied to improve customer experience and automate business processes.
